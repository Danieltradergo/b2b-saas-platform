# B2B SaaS Platform - Guia de Configuracao\n\n## Visao Geral do Projeto\n\nSistema B2B SaaS completo para empresas brasileiras com:\n- Landing page com tabela de precos\n- Sistema de autenticacao via Supabase\n- Painel de usuario\n- Painel de administracao\n- Gerenciamento de planos e usuarios\n- Interface responsiva com Tailwind CSS\n- Totalmente gratis com tecnologias open source\n\n## Arquitetura\n\n### Frontend\n- Next.js 14 (React Framework)\n- Tailwind CSS\n- TypeScript\n- Autenticacao: @supabase/auth-ui-react\n\n### Backend\n- Supabase (PostgreSQL + Auth)\n- API REST\n- Real-time Subscriptions\n\n### Deploy\n- Vercel (Frontend)\n- Netlify (Dominio/DNS alternativo)\n- Supabase (Backend)\n\n## Estrutura de Diretorios\n\n```\nb2b-saas-platform/\n├── app/\n│   ├── page.tsx                    # Landing page\n│   ├── pricing/\n│   │   └── page.tsx               # Pagina de precos\n│   ├── auth/\n│   │   └── page.tsx               # Pagina de login/registro\n│   ├── dashboard/\n│   │   ├── page.tsx               # Painel do usuario\n│   │   ├── companies/             # Gerenciar empresas\n│   │   ├── invoices/              # Faturas\n│   │   └── settings/              # Configuracoes\n│   └── admin/\n│       ├── page.tsx               # Painel admin\n│       ├── users/                 # Gerenciar usuarios\n│       ├── plans/                 # Gerenciar planos\n│       └── analytics/             # Analiticas\n├── components/\n│   ├── Navbar.tsx\n│   ├── Footer.tsx\n│   ├── PricingTable.tsx\n│   ├── FeatureCard.tsx\n│   └── AuthForm.tsx\n├── lib/\n│   ├── supabase.ts               # Configuracao Supabase\n│   ├── auth.ts                   # Logica de autenticacao\n│   └── api-client.ts             # Cliente HTTP\n├── styles/\n│   └── globals.css\n├── public/                         # Assets\n├── package.json\n├── tailwind.config.js\n├── tsconfig.json\n└── .env.local                      # Variaveis de ambiente\n```\n\n## Configuracao Local\n\n### 1. Clonar o repositorio\n```bash\ngit clone https://github.com/Danieltradergo/b2b-saas-platform.git\ncd b2b-saas-platform\n```\n\n### 2. Instalar dependencias\n```bash\nnpm install\n```\n\n### 3. Configurar variaveis de ambiente\nCrie um arquivo `.env.local` com:\n```\nNEXT_PUBLIC_SUPABASE_URL=seu_supabase_url\nNEXT_PUBLIC_SUPABASE_ANON_KEY=sua_anon_key\nSUPABASE_SERVICE_ROLE_KEY=sua_service_role_key\n```\n\n### 4. Executar localmente\n```bash\nnpm run dev\n```\nAcesse em `http://localhost:3000`\n\n## Tabela de Precos\n\n### Plano Basic - R\$99/mes\n- Ate 5 usuarios\n- 10GB armazenamento\n- Suporte por email\n- Relatorios basicos\n\n### Plano Professional - R\$299/mes\n- Ate 50 usuarios\n- 100GB armazenamento\n- Suporte prioritario\n- Relatorios avancados\n- API customizada\n\n### Plano Enterprise - Consulte\n- Usuarios ilimitados\n- Armazenamento ilimitado\n- Suporte dedicado 24/7\n- Integracao customizada\n- SLA garantido 99.9%\n\n## Tabela de Banco de Dados\n\n### users\n- id (UUID, PK)\n- email (TEXT, UNIQUE)\n- password_hash (TEXT)\n- full_name (TEXT)\n- phone (TEXT)\n- company_id (UUID, FK)\n- role (ENUM: admin, user)\n- created_at (TIMESTAMP)\n- updated_at (TIMESTAMP)\n\n### companies\n- id (UUID, PK)\n- name (TEXT)\n- cnpj (TEXT, UNIQUE)\n- email (TEXT)\n- phone (TEXT)\n- address (TEXT)\n- plan_id (UUID, FK)\n- status (ENUM: active, inactive, suspended)\n- created_at (TIMESTAMP)\n- updated_at (TIMESTAMP)\n\n### pricing_plans\n- id (UUID, PK)\n- name (TEXT)\n- price (NUMERIC)\n- billing_period (ENUM: monthly, semiannual, annual)\n- max_users (INTEGER)\n- storage_gb (INTEGER)\n- features (JSONB)\n- created_at (TIMESTAMP)\n\n### subscriptions\n- id (UUID, PK)\n- company_id (UUID, FK)\n- plan_id (UUID, FK)\n- status (ENUM: active, cancelled, suspended)\n- next_billing_date (DATE)\n- created_at (TIMESTAMP)\n- updated_at (TIMESTAMP)\n\n## Deploy no Vercel\n\n1. Fazer push do codigo para GitHub\n2. Conectar repositorio no Vercel\n3. Configurar variaveis de ambiente\n4. Deploy automatico ao fazer push\n\n## Deploy no Netlify (Dominio)\n\n1. Conectar dominio no Netlify\n2. Configurar DNS\n3. Redirecionar para Vercel\n\n## Funcionalidades Principais\n\n### Landing Page\n- Hero section com CTA\n- Showcase de features\n- Tabela de precos\n- Testimonials\n- FAQ\n- Footer com links\n\n### Autenticacao\n- Registro de empresas\n- Login seguro\n- Recuperacao de senha\n- 2FA opcional\n\n### Dashboard Usuario\n- Visao geral da empresa\n- Listagem de usuarios\n- Gerenciar assinatura\n- Invoices/Faturas\n- Configuracoes da conta\n\n### Admin Dashboard\n- Listagem de empresas\n- Gerenciar usuarios globais\n- Gerenciar planos\n- Analiticas e relatorios\n- Suporte a clientes\n\n## Proximos Passos\n\n1.
